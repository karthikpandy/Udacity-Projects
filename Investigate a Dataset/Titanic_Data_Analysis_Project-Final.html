#Function to get the Salutation separately
def Salutation(df):
    """To define the Salutation separately"""
    if 'Mrs.' in df:
        return 'Mrs'
    elif 'Miss.' in df:
        return  'Miss'
    elif 'Master.' in df:
        return  'Master'
    else:
        return 'Mr'

# Funtion to add new column new_age, with median replaced for blanks
def new_age (Age, Salutation):
    """To define the Salutation separately"""
    if Age == 0.0:
        if Salutation == 'Master':
            return titanic_df['Age'][titanic_df['Age'] != 0.0][titanic_df['Salutation'] == 'Master'].median()
        if Salutation == 'Mrs':
            return  titanic_df['Age'][titanic_df['Age']!= 0.0][titanic_df['Salutation'] == 'Mrs'].median()
        if Salutation == 'Miss':
            return  titanic_df['Age'][titanic_df['Age']!= 0.0][titanic_df['Salutation'] == 'Miss'].median()
        if Salutation == 'Mr':
            return  titanic_df['Age'][titanic_df['Age']!= 0.0][titanic_df['Salutation'] == 'Mr'].median()
    else:
        return Age

#Function to add column Survival to describe value of Survived
def survival (survival):
    """add column to define survival value"""
    if survival['Survived'] == 0:
        return 'Did not Survived'
    else:
        return 'Survived'

Conclusion:¶

  • Based on our analysis on the sample data, it seems like Survival rate is less for Male Passengers,Lower class and Passengers travelling alone

  • Child and Females in PClass 1 and 2 are most likely to be survived

  • As this a sample set of data, there may be a slight difference for the whole data, we only have around 70% of total data

  • As we saw initially most of the Cabin data are not available, and from our other analysis Passenger Class 3 has less survival rate. So if we had much data in Cabin it will give opportunities to analyze more in Pclass 3 itself and get some inisights

  • Also there were some Age data missing which we filled wiith Median, it can make a small impact if we had all Age data
